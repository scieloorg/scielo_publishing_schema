deceased

Deceased

Indicates whether or not the contributor described in the element was deceased when the document or document component was published.

Attribute Values

In Element

<contrib>
ValueMeaning
noThe contributor was living when the document was published.
yesThe contributor had died by the time the document was published.
Restriction: This is an optional attribute; there is no default.
