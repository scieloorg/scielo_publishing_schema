y-size

Y Size of the Glyph

The number of rows in the bit-mapped glyph that has been declared with a <glyph-data> element as part of a <private-char> element.

Attribute Values

In Element

<glyph-data>
ValueMeaning
NumberThe number of rows (for example, “25”) used to define the glyph in a <glyph-data> element.
Restriction: This is an optional attribute; there is no default.
