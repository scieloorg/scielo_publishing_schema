%title-elements;

Elements Used in Titles

Elements that can be mixed with the data characters inside the various title elements, for example, inside <title>, <subtitle>, <trans-title>, etc.

Remarks

Historical Note: In earlier versions of this Tag Set, this parameter entity was used directly with the elements names above. However, in NISO JATS Version 1.0 unnecessary dependencies have been removed; each element was given its own element-naming parameter entity. In the default case, these call this parameter entity for backwards compatibility.
