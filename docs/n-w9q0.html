x-size

X Size of the Glyph

The pixels per row in the bit-mapped glyph that has been declared with a <glyph-data> element as part of a <private-char> element.

Attribute Values

In Element

<glyph-data>
ValueMeaning
A numberAn integer number of pixels, for example, “45”, that is the number of pixels in each row of the bitmap of the <glyph-data> element.
Restriction: This is an optional attribute; there is no default.
