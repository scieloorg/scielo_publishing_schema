.. _elemento-given-names:

<given-names>
=============

Aparece em:

  :ref:`elemento-name`

Ocorre:

  Zero ou mais vezes


Identifica o prenome do autor, ou seja, todos os nomes exceto pelo sobrenome.

.. code-block:: xml

    ...
    <name>
        <surname>Santos</surname>
        <given-names>Ana Maria da Silva</given-names>
        ...
    </name>
    ...
