calendar

Calendar

Name of the calendar used in naming a date such as a publication date, for example, “Gregorian”, “Japanese” (Emperor years), or “Islamic”.

Attribute Values

In Elements

<conf-date>, <date>, <date-in-citation>, <pub-date>, <year>
ValueMeaning
Text, numbers, or special characters.Name of the calendar used in the date, for example, “Gregorian”.
Restriction: This is an optional attribute; there is no default.
