vol

Volume of Related Article

Volume of the journal in which a related journal article exists.

Usage

The @vol attribute can be used, along with the @journal-id, @issue, and @page attributes, to provide the metadata for a related journal article.

Attribute Values

In Element

<related-article>
ValueMeaning
Text, numbers, or special charactersThe journal volume number for a related journal article, for example, “3” for the third volume, “125” for the one hundred and twenty-fifth, or “XV” for the fifteenth.
Restriction: This is an optional attribute; there is no default.
