NISO JATS XHTML-based Table Setup Module

Module that defines the elements in the NISO JATS table model, which is based on and intended to be converted easily to the XHTML table model.

Remarks

This module, named in the base Suite Module to Name the Modules (%modules.ent;), should be called by a DTD to set up the parameter entities needed by the NISO JATS XHTML-inspired table model, to allow the NISO JATS table elements to be used in this Tag Set.
