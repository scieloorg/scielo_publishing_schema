%label.class;

Label Class Elements

Class comprising the <label> element. A <label> can be used to hold the pre-generated or author-provided number or symbol of a labeled object such as a footnote (“3” or “*”) or a table wrapper (“Table 6”)

Remarks

The Label Class Elements parameter entity was created as a means of constraining use of the <label> element in the base Suite and derived Tag Sets.
