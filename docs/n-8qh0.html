count

Count

Number of specific objects that appear in the document.

Remarks

This attribute names the number of objects. The type of objects being counted may be indicated either by the element name on which this attribute occurs (for example, <page-count> says that the count is of pages) or, for the <count> element, by the value of the @count-type attribute.

Attribute Values

In Elements

<count>, <equation-count>, <fig-count>, <page-count>, <ref-count>, <table-count>, <word-count>
ValueMeaning
Counting numberThe number of objects being counted, for example, “5” for five tables or five figures.
Restriction: This attribute must be specified if the element is used.
