XHTML 1.1 table model

NISO JATS table modules, which is based on and intended to be converted easily to the XHTML table model.

Remarks

External module containing the NISO JATS XHTML-inspired table model; this module is named in the Module to Name the Modules (%modules.ent;) and should then be called from the NISO JATS Table Setup Module (%XHTMLtablesetup.ent;) to include XHTML 1.1 table elements in the Tag Set.
