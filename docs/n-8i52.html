%simple-phrase;

Simple Phrase-Level Textual Elements

Names elements that may be used almost anywhere text is used, for example, inside most kinds of titles

Remarks

This parameter entity is slightly more complex and contains more elements than the parameter entity %simple-text;; this parameter entity would be used for potentially complex models.
