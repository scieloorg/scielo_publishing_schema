%style.attrib;

Style Attribute (NISO JATS table model)

Defines the @style attribute, its values, and default for use in attribute lists in NISO JATS table model (based on and designed to be converted easily to the XHTML 1.1 table model).

Remarks

As part of a NISO JATS table, the attribute should be used to support inline style markup

Since this parameter entity has not been modified from the XHTML 1.1 Tag Set, it does not follow the naming conventions for this Tag Set: the separator character is a period, not a hyphen.

Content Model

<!ENTITY % style.attrib
     "style        CDATA                    #IMPLIED"
>

Module

JATS-xhtml-inlstyle-1.mod
