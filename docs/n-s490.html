%caption.attlist;

Table caption attributes

This parameter entity holds the INCLUDE or IGNORE keyword to control a conditional marked section. This Tag Set has redefined the attribute list for the element <caption>. Therefore, the NISO JATS table model modified the standard XHMTLâ„¢ attribute list call to change an INCLUDE to an IGNORE, effectively removing the default XHTML 1.1 definition of the caption attributes, so the attributes would not be multiply defined.

Remarks

To change the attributes of the <caption> element, use %caption-atts;.

This parameter entity names the attributes for the element <caption>; the parameter entity %caption.attlist; merely sets an INCLUDE/IGNORE switch to turn the NISO JATS XHTML-inspired table model default attributes of <caption> on or off.
