rules

Rules (NISO JATS table model)

Defines which rules to draw between the rows and columns of an XHTML 1.1 table. (This attribute is based on and intended to be converted easily to the XHMTL rules attribute.)

Remarks

If the @rules attribute has not been used, the presence or absence of rules will be based on the @border attribute. If 

<th border="0">

or there is no @border attribute, then no rules are drawn. Otherwise, the value of the @rules attribute is assumed to be “all”. (The related table attribute @frame only concerns rules around the outside of the table.)

Attribute Values

In Element

<table>
ValueMeaning
allRules on all rows and columns.
colsRules between columns.
groupsRules between groups.
noneNo rules in table.
rowsRules between rows.
Restriction: This is an optional attribute; there is no default.
